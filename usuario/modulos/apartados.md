## ¿Qué es un apartado?

Un apartado reserva uno o más productos para un cliente que paga un anticipo.
El producto queda bloqueado del inventario — no puede venderse a otro cliente
mientras el apartado esté activo. El cliente paga el resto antes de llevárselo.

## Diferencia entre apartado y crédito

| | Apartado | Crédito |
|---|---|---|
| ¿Cuándo sale el producto? | Al liquidar | Inmediatamente |
| ¿Quién paga primero? | El cliente | El negocio fía |
| Stock reservado | Sí | No |

## Crear un apartado

1. Ve a **Apartados** → **"Nuevo apartado"**.
2. Busca el producto y selecciona la variante si aplica.
3. Ingresa los datos del cliente (nombre y teléfono).
4. Define el anticipo inicial y la fecha límite.
5. Confirma — el stock queda reservado y el anticipo registrado.

## Registrar un abono

Cuando el cliente viene a pagar una parte:

1. Busca el apartado en la lista de **Abiertos**.
2. Haz clic en el apartado → **"Registrar abono"**.
3. Ingresa el monto y el método de pago.
4. Guarda — el saldo pendiente se actualiza.

## Liquidar el apartado

Cuando el cliente paga el total:

1. Abre el apartado y haz clic en **"Liquidar"**.
2. Cobra el saldo restante.
3. Confirma — se genera la venta, el inventario se descuenta y el cliente
   puede llevarse el producto.

## Cancelar un apartado

Solo el dueño o gerente puede cancelar un apartado. El stock reservado
se libera automáticamente al cancelar.

## Preguntas frecuentes

**¿El anticipo aparece en el corte de caja?**
Sí, como "Abonos de apartados" — separado de las ventas para no generar
doble conteo. La venta real se registra al liquidar.

**¿Qué pasa si el cliente no regresa antes de la fecha límite?**
El apartado muestra una alerta de vencimiento. El dueño decide si extender
el plazo o cancelar y liberar el stock.

**¿Puedo apartar un producto sin stock?**
Sí, marcando la opción de forzar el apartado — útil para mercancía en camino
que aún no ha llegado al inventario.
